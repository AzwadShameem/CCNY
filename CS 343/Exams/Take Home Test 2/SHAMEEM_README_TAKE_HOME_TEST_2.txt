The code used in MARS Simulator is in the MIPS folder.
You may open MARS Simulator and run the Shameem_gcd.asm file to
see the results and execution for youself.

The code used in Visual Studio is the Visual Studio folder.
You may open Visual Studio and run Shameem_gcd.c file 
to see the results and execution for yourself.

The code used in the Linux GCC and GDB is in the Linux folder.
You may open the terminal and go into the Linux folder and
use gcc to compile the Shameem_gcd.c file accordingly and then use it
with gdb to see the results and execution for yourself.
